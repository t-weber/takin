Additional Set-up Steps

Some of Takin's program modules optionally require Gnuplot. It is especially recommended in order to have a live-display of the progress in the convolution fitter. The steps to correctly install Gnuplot and its graphical output terminals are given as follows (pick the package manager which applies for your system):

  • APT: 
    	apt-get install gnuplot gnuplot-qt
  • DNF: 
    	dnf install gnuplot
  • YUM: 
    	yum install gnuplot
  • Pacman: 
    	pacman -S gnuplot
  • Homebrew: 
    	brew install gnuplot --with-qt --with-x11
